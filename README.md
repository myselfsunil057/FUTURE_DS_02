# 📊 Customer Churn Analysis Dashboard

---

## 🧠 Project Overview

This project presents an interactive **Customer Churn Analysis Dashboard** built using Power BI to analyze customer retention, churn behavior, and user engagement patterns across multiple dimensions such as device type, subscription plans, network quality, and behavioral signals.

The objective is to convert raw user-level data into **actionable retention intelligence** and identify the key drivers behind customer churn.

---

## 🎯 Objective

* Analyze overall churn rate and retention performance
* Identify key behavioral and technical drivers of churn
* Evaluate device, network, and buffering impact on retention
* Understand subscription plan effectiveness and user lifecycle behavior
* Support data-driven retention strategy and decision-making

---

## 📊 Key Metrics

* 👥 Total Users: 4,600
* ✅ Active Users: 3,188
* ❌ Churned Users: 1,412
* 📉 Churn Rate: 30.70%

---

## 📌 Dashboard Features

### 🧾 KPI Summary

* Total Users, Active Users, Churned Users, Churn Rate
* High-level retention snapshot with clear KPI cards

### 📱 Device Analysis

* Churn comparison across Android, iOS, and Web
* Identification of high-risk platforms

### 🌐 Network Impact Analysis

* 3G / 4G / 5G / WiFi-based churn segmentation
* Relationship between network quality and retention

### ⏳ Performance Analysis

* Buffering time vs churn correlation
* Identification of latency-driven user drop-off

### 💳 Subscription Analysis

* Free vs Premium vs Family plan churn comparison
* Plan-wise retention effectiveness

### 👤 User Segmentation

* Age group and gender-based churn distribution
* Identification of high-risk user cohorts

### 🔁 Behavioral Insights

* Auto-renewal impact on churn probability
* Payment method influence on retention

---

## 📊 Key Business Insights

* Overall churn rate is **high (30.70%)**, indicating retention inefficiency
* Mobile users (Android & iOS) show higher churn than web users
* Buffering time is a **strong churn predictor**
* Non-auto-renew users are significantly more likely to churn
* Premium users show relatively better retention stability
* Core engagement is concentrated in the 18–35 age group

---

## ⚠️ Business Risks Identified

* High dependency on mobile user experience quality
* Strong correlation between buffering and churn risk
* Weak retention among non-auto-renew users
* Performance issues directly impacting customer lifecycle

---

## 🚀 Business Opportunities

* Optimize mobile app performance (especially Android)
* Reduce buffering time to improve retention
* Increase auto-renew adoption through incentives
* Strengthen Premium upgrade strategy for retention gains
* Build predictive churn detection models
